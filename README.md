## Overview
NewsBot 2.0 is a comprehensive NLP system designed to analyze news articles using:
- Advanced content classification
- Topic modeling (LDA + NMF)
- Sentiment evolution analysis
- Named Entity Recognition
- Multilingual analysis (detection + translation)
- Semantic search using embeddings
- Conversational interface with intent detection
- Insight generation using language models

This project builds upon the Midterm NewsBot and expands it into a production-ready system.
